react-native-示例代码浅析

发表于 2016-01-20   |   分类于 react-native   |  

##写在前面的话
ES6让我想起丢弃多年的JAVA代码

严格模式

'use strict';

ES6中默认开启

##引入所使用到组件

ES5 Code

var React = require('react-native');
var {
  AppRegistry,
  StyleSheet,
  Text,
  View,
} = React;

以上解构赋值代码等同于

ES6 Code

import React, {
  AppRegistry,
  StyleSheet,
  Text,
  View,
} from 'react-native';

##创建一个组件
创建一个用于渲染内容的组件,return里面就是JSX代码了

//ES5 Code
var Demo = React.createClass({
  render: function() {
    return (
      <View style={styles.container}>
        <Text style={styles.welcome}>
          Welcome to React Native Demo!
        </Text>
        <Text style={styles.instructions}>
          To get started, edit index.ios.js
        </Text>
        <Text style={styles.instructions}>
          Press Cmd+R to reload,{'\n'}
          Cmd+D or shake for dev menu
        </Text>
      </View>
    );
  }
});

以上代码等同于

//ES6 Code
class Demo extends React.Component {
    render() {
      return (
        <View style={styles.container}>
          <Text style={styles.welcome}>
            Welcome to React Native Demo!
          </Text>
          <Text style={styles.instructions}>
            To get started, edit index.ios.js
          </Text>
          <Text style={styles.instructions}>
            Press Cmd+R to reload,{'\n'}
            Cmd+D or shake for dev menu
          </Text>
        </View>
      );
    }
}

##创建组件的样式
这里是调用api,没有ES6的写法了,这里主要了解下容器的布局,又回忆起ExtJS的布局

var styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  welcome: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
  instructions: {
    textAlign: 'center',
    color: '#333333',
    marginBottom: 5,
  },
});

##代码入口
AppRegistry.registerComponent(‘Demo’, () => Demo);

react-native环境搭建

发表于 2016-01-12   |   分类于 react-native   |  

React Native如此之火爆,不耍下不是好前端:)

快速开始

搭建iOS开发环境,暂不涉及到搭建Android平台

安装

我的node环境是通过nvm安装,node版本v4.1.2(最好将npm的镜像切换成taobao)

安装watchman,facebook提供的文件监视器,便于文件编译(本章节没有用到)

安装React Native CLI工具,可能需要sudo权限

安装没有出错,继续,出错请google之.

创建Demo

可能需要sudo

执行命令后,需要稍等一会,因为需要联网下载,成功后会有如下信息展示

Installing react-native package from npm...
Setting up new React Native app in /Users/your path/workspace/react_native_demo/Demo
To run your app on iOS:

Open /Users/your path/workspace/react_native_demo/Demo/ios/Demo.xcodeproj in Xcode

Hit the Run button

To run your app on Android:

Have an Android emulator running (quickest way to get started), or a device connected
cd /Users/your path/workspace/react_native_demo/Demo

react-native run-android

按上面提示iOS用XCode打开就可以了,Android则需要使用模拟器

Android开发目录结构:

入口文件:

index.android.js

iOS开发目录结构:

Demo - 工程源文件目录

Demo.xcodeproj - 这个文件夹有什么作用,apple官方居然没有说明

网上找到大概说明如下,供 参考

DemoTests - 测试目录

入口文件:

index.ios.js

打开”Demo.xcodeproj”时,可能会提示你没有权限,检查下”/Users/your path/.babel.json”该文件是否

可读写

加上权限

编译时会新开个终端窗口,通过后会启动模拟器
